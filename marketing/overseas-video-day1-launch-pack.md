# Vancine Overseas Video Day 1 Launch Pack

目标：今天完成 TikTok + YouTube Shorts 的第一轮测试。不要追求完美，先验证哪个卖点能带来海外开发者点击。

当前正确定位：

```text
Vancine helps global developers access top Chinese AI models through one unified API.
```

不要使用的口径：

```text
One API for GPT, Claude, Gemini, and Chinese AI models.
```

原因：Vancine 当前主打中国顶尖模型，不应暗示已接入海外闭源模型。

公开文案使用：

```text

```

不要公开写：

```text

```

建议后台策略：

- 普通自然流量注册送 `0.10-0.30 USD` 等值额度
- 每日请求次数限制
- 单次 token / 图片 / 视频时长限制
- 邮箱验证后领取
- 高质量用户人工提高额度

## 5. 今天的视频制作方式

最快做法：

1. 用 CapCut / 剪映国际版创建 9:16 项目。
2. 使用 Vancine 渐变背景或官网截图做背景。
3. 放大标题，保证手机上可读。
4. 用英文 AI 语音或自己录音。
5. 录制 5-10 秒 Vancine 文档/API 页面作为产品画面。
6. 结尾放 Vancine logo 和网址。
7. 导出 1080x1920 MP4。

不要做：

- 不要搬运别人完整视频
- 不要去水印
- 不要暗示 Vancine 可调用 GPT/Claude/Gemini
- 不要把试用额度写成固定 `$1`

## 6. 发布后 24 小时复盘

每条视频记录：

```text
Platform:
Post URL:
Topic:
Hook:
Views 3h:
Views 24h:
Likes:
Comments:
Profile clicks:
Website clicks:
Signups:
API keys created:
First API calls:
Notes:
Decision: kill / iterate / double down
```

判断：

- 播放高但点击低：内容太泛，要更开发者化
- 播放低但点击高：题材精准，继续做
- 注册多但 API call 少：落地页或 docs 需要优化
- 评论问价格/模型/API：值得做下一条视频
