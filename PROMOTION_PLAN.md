# Vancine 推广方案

## 一、产品定位

**一句话描述**：统一的 AI 模型聚合与分发网关，一个 API 接入全球主流 AI 模型

**核心价值**：
- 开发者无需对接多个供应商
- 统一 OpenAI 兼容接口
- 按量付费，无订阅费用
- 国产模型价格仅为国际模型的 20-50%

**目标用户**：
- 独立开发者
- AI 应用创业者
- 小型技术团队
- 正在使用多个 AI API 的开发者

**V2EX / 掘金标题**：
> 分享一个我做的 AI 模型聚合网关，一个 API 调用 200+ 模型

**小红书标题**：
> 程序员省钱神器！国产 AI 模型价格只有 OpenAI 的 20%

**Twitter 标题**：
> 🚀 Built an AI model hub - one API for 200+ models from OpenAI, Claude, Gemini & more. Pay-as-you-go, no subscription. Try free!

### 2. 产品亮点

1. **统一接口** — OpenAI 兼容，无需改代码
2. **价格优势** — 国产模型价格低至国际模型的 20%
3. **按量付费** — 无订阅，无最低消费
4. **多模态支持** — 文本、图片、视频、音频、3D
5. **企业级** — 完整的日志、监控、权限管理

### 3. 目标话术

**对独立开发者**：
> 「还在为对接多个 AI API 烦恼吗？Vancine 让你一个接口搞定所有模型，还能省钱。」

**对创业者**：
> 「AI 应用开发成本太高？试试 Vancine，国产模型价格只有国际模型的 20%，按量付费无风险。」

**对技术团队**：
> 「统一的 AI 网关，完整的日志和权限管理，让团队协作更高效。」
